Kotlin环境搭建

创建Android工程

安装Kotlin插件

File -> Setting -> Plugins 输入Kotlin

点击右边的安装，重启Android Studio 就可以了

把Java转换为Kotlin

鼠标选中要转换的类，点击Code -> Convert Java File to Kotlin File 或者直接快捷键Ctrl + Alt + Shift + K

转换好了，运行下就会出现

点击设置Configure

点击Sync Now 这样就好了，所有的都会自动配置上。
