新建分类

在命令行里面输入：

hexo new page “about”
然后你会发现source里面多了个目录about，里面有个index.md。其实你也可以手动建立。页面的格式和文章一样。

接着把链接加上，themes//_config.yml里面的menu一项，添加一行About: /about。

完事。
